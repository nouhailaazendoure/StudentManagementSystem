# Student Management System

**Java Academic Project** – Console application to manage students.

---

## Overview

This project allows basic management of students in a console application:

- Add a student
- Modify student information
- Remove a student
- Display all students
- Sort students by name or average
- Search for a student by ID
- Display top and bottom students
- Count students who passed (average ≥ 10)

Technologies used: Java, ArrayList, Scanner

---

## Features

- Console-based user interface
- Fully implemented add, modify, remove, display functionalities
- Sorting and searching features
- Handles variable number of grades for each student
